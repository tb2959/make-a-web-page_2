Lesson 1

HTML

The primary type of documents on the internet are HTML. HTML (Hypertext Markup Language) documents are a tree-like structure made up of elements. These elements have a beginning and ending tag; of which, can be text, spans, paragraphs, etc. So you can have elements within other elements.

TAGS

Tags make up elements and allow you to affect the output of the code that is written. You can turn words into links to other websites or even images. There are two types of tags inline and block. Inline allows the code to run continuously without structure around it. Whereas block contains the content within a virtual box with adjustable height and width.

CSS

CSS compliments HTML by providing style to the elements. These can include text & background color, alignment, height and width.

Avoiding Repetition

Avoiding repetition is important to avoid rework and errors. Applying the same style to multiple elements can mean having to reapply it over and over. Attention to detail when making the changes is crucial. If the overall design requires a consistent format, the programmer can assign the same div class to each element and make the changes simultaneously.
